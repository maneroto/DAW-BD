¿Qué es entonces una base de datos - DB?

Es una colección de archivos interrelacionados de manera lógica por la información y su descripción, diseñadapara atender las necesidades de la gestión de información de alguna entidad u organización.

¿En qué casos es conveniente usar bases de datos?

En todo momento que se requiera de almacenar información que esté relacionada con más información. Cuando se requiera alamacenar información, pero que esté libre a que pueda sufrir cambios y que se le puedan agregar o eliminar características/fields/atributos. Cuando ocupemos el tener nuestra información organizada.

¿Qué es un sistema de gestión de base de datos - DBMS?

Es un software que trabaja sobre una base de datos y que permite el que un usuario defina, cree, modifique, mantenga y controle el acceso a esta base de datos. Por lo mismo es que un DBMS facilita y simplifica el acceso a los datos.

Enlista y explica las funciones/responsabilidades que tienen los DBMS.

Un DBMS debe de permitir que el usuario:

  • Defina
  • Cree
  • Interactúe
  • Mantenga
  • Modifique
  • Controle
  • Recupere
  • Actualice
  • Almacene
